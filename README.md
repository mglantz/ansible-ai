# ansible-ai
A project which aims to test Ansible with different AI agents.

Currently it tests Claude Sonnet 4.6 and Chat GPT 4o.

To view tests, see [https://mglantz.github.io/ansible-ai](https://mglantz.github.io/ansible-ai)

## Test prompt
Create an Ansible playbook which installs Apache web server on Red Hat Enterprise Linux. The web server should serve an index.html page which says "Hej, hallo, hello, hi."
